== Installation  ==

1. Unpack the *.zip file and extract the /scroll_announcement/ folder.
2. Drop the ‘scroll_announcement’ folder into your ‘wp-content/plugins’ folder
3. In word press administration panels, click on plug-in from the menu.
4. You should see your new ‘scroll_announcement’ plug-in listed under Inactive plug-in tab.
5. To turn the word presses plug-in on, click activate.
6. Go to widget link under Appearance tab, Drag & drop scroll_announcement widget to your desired location in the active sidebar.

you have ten options ,i.e 10 announcement’s for the people to see.

you can style the announcement using inline css.

please use <br> tag for next line.
== Frequently Asked Questions == 

I installed the plugin but can’t see it. What’s wrong?
Make sure the directory name is entered correctly on the settings page.

Widget title appears but no announcements?
This may be due to pressing of enter button or any html errors.
